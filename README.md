# 博客项目

## 开发前准备

* `yarn install` 安装依赖


## 目录

```text
.
├── App.test.tsx
├── App.tsx
├── api          // api定义
├── assets       // 静态资源文件
├── components   // 业务组件
├── constants    // 常量定义
├── history.ts
├── index.tsx
├── mock         // mock脚本
├── public-lib   // 公共函数/组件
├── react-app-env.d.ts
├── routes       // 页面
├── serviceWorker.ts
├── styles       // 公共样式 or 全局样式
└── types        // 系统常用变量类型定义
```
